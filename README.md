# Hermes Agent - Front-End Clone (Educational Purpose)

Proyek ini adalah latihan *slicing* dan pengembangan front-end untuk mereplikasi tampilan web Hermes Agent. Semua kode (HTML, CSS, dan JavaScript) ditulis dari awal secara mandiri untuk keperluan pembelajaran.

## Sumber Aset Gambar
* **Aset Visual:** Gambar dan elemen grafis yang digunakan dalam proyek ini diambil langsung dari situs resmi Hermes Agent untuk menjaga akurasi tampilan selama proses latihan.
